
# Rubric (Out of 70)

## Interface Design (10)
- Clean layout & styling (Bootstrap or custom)
- Accessible labels & readable text

## Controlled Form Components (15)
- `useState` controls inputs
- Real-time updates & reset after submit

## Validation (15)
- All fields required
- Price: number with up to 2 decimals, >= 0
- Stock: non-negative integer
- Inline feedback + console log

## Dynamic Display (15)
- Reusable components
- Product list updates after add/delete
- Empty-state message

## Persistence (10)
- Correct use of localStorage
- Data persists on reload
- Delete updates storage & UI

## Code Quality (5)
- Clear naming, ES6 syntax, consistent formatting
